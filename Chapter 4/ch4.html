Rules for Naming JS Variables

The variable declaration results are visible in the **browser console** (Press F12 to view).

  • Variable names can only contain **letters**, **numbers**, **underscores** (`_`), and **dollar signs** (`$`).
    For example: $my_1stVariable
  • Variables must begin with a **letter**, **underscore** (`_`), or **dollar sign** (`$`).
    For example: $name, _name or name
  • Variable names are case **sensitive**. (total is different from Total)
  • Variable names should not be JS **reserved keywords** (e.g., `for`, `let`, `class`).
